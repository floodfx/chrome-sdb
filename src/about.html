About

Simple DB Tool v0.2

This tool is built for Simple DB developers everywhere! Please use it and provide feedback.

New tool replacing old tool

This is a new tool that replaces the old Firefox plugin tool ( Sdbtool ) that I wrote a few years back and which has been a PITA to keep updated as Firefox started releasing new updates every couple of weeks and required updating the plugin... Plus, Chrome is just faster than Firefox. :)

Open Source

Chrome Simple DB Tool is open source. Pull requests are happily accepted!

Chrome Store

We also hope to get this listed in the Google Chrome store...eventually.
